# AllMyPrompts
easily save and store all prompts I've used or want to use


# Feature
1. save prompts. You can highlight and right click to save prompts.
2. check saved prompts. You can see all your saved prompts in chrome extension.
3. tag saved prompts. You can add a tag to saved prompts so that they are categorized. 
4. copy and use saved prompts. You can click saved prompts in AllMyPrompts chrome extension to copy prompts.
5. rank prompts use. AllMyPrompts will automatically counts the number of times you copied a prompt. top copied prompts will be displayed at the top of chrome extension window as "frequently used prompts".
